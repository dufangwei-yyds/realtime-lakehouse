原始部署笔记中出现了数据库密码、临时密码和开放式 root 配置。

- `${MYSQL_ROOT_PASSWORD}`
- `${HIVE_METASTORE_PASSWORD}`
- `${SUPERSET_DB_PASSWORD}`

并增加以下约束：

- 测试环境账号和生产环境账号严格隔离
- 不在文档中写明真实口令
- 配置文件优先采用环境变量或外置密钥文件

## 5.2 Hive Metastore 部署基线

### 5.2.1 作用

Hive 在当前测试环境中主要承担：

- Hive Metastore 服务
- Spark / Flink / Paimon 元数据集成支撑

### 5.2.2 核心配置建议

`hive-site.xml` 中需要重点确认：

- MySQL 元数据库连接 URL
- Driver 类名
- 用户名与密码
- Metastore 端口
- `hive.metastore.uris`

建议正式文档写法如下：

```xml
<property>
  <name>javax.jdo.option.ConnectionURL</name>
  <value>jdbc:mysql://192.168.63.128:3306/hive?serverTimezone=Asia/Shanghai</value>
</property>
<property>
  <name>javax.jdo.option.ConnectionDriverName</name>
  <value>com.mysql.cj.jdbc.Driver</value>
</property>
<property>
  <name>javax.jdo.option.ConnectionUserName</name>
  <value>${HIVE_META_USER}</value>
</property>
<property>
  <name>javax.jdo.option.ConnectionPassword</name>
  <value>${HIVE_META_PASSWORD}</value>
</property>
<property>
  <name>hive.metastore.uris</name>
  <value>thrift://192.168.63.128:9083</value>
</property>
```

### 5.2.3 启停命令

```bash
schematool -dbType mysql -initSchema
hive --service metastore
hiveserver2
```

### 5.2.4 验证项

- Metastore 端口 `9083` 可连通

- `mysql-connector-j-*`

  bind-host: 0.0.0.0
  host: 192.168.63.128
  numberOfTaskSlots: 4

  default: 1

  backend: hashmap
  checkpoints:
    dir: hdfs://192.168.63.128:8020/ckps
  savepoints:
    dir: hdfs://192.168.63.128:8020/svps

rest:
  address: 192.168.63.128
  bind-address: 0.0.0.0
  port: 8081

historyserver:
  web:
    address: 0.0.0.0
    port: 8082
  archive:
    fs:
      dir: hdfs://192.168.63.128:8020/completed-jobs/

fi

# 检查进程是否运行

check_process() {
local process_name=$1
    if $JPS_CMD -l 2>/dev/null | grep -q "$process_name"; then
return 0
fi
return 1
}

# 等待进程启动

wait_for_process() {
local process_name=$1
local max_wait=$2
local waited=0

    print_info "等待 $process_name 启动..."
    while [ $waited -lt $max_wait ]; do
        if check_process "$process_name"; then
            print_success "$process_name 启动成功"
            return 0
        fi
        sleep 5
        waited=$((waited + 5))
        echo -n "."
    done
    echo ""
    print_error "$process_name 启动超时"
    return 1

}
