## 一、文档说明

本清单用于统一记录测试环境中的服务、端口、协议、用途和验证方式，主要服务于：

- 环境部署核对
- 联调前连通性检查
- 问题排查
- 项目验收

当前测试环境核心节点：

- `192.168.63.128`

## 九、推荐验证命令

### 9.1 端口监听检查

```bash
ss -lntp
```

定向检查：

```bash
ss -lntp | grep 9092
ss -lntp | grep 8081
ss -lntp | grep 9030
ss -lntp | grep 8888
```

### 9.2 Web 页面访问检查

建议验证以下页面：

- `http://192.168.63.128:9870`
- `http://192.168.63.128:18088`
- `http://192.168.63.128:8081`
- `http://192.168.63.128:8082`
- `http://192.168.63.128:7030`
- `http://192.168.63.128:7040`
- `http://192.168.63.128:8888`
- `http://192.168.63.128:9090`

### 9.3 SQL / 命令行连通性检查

MySQL：

```bash
mysql -uroot -h192.168.63.128 -P3306
```

Doris：

```bash
mysql -uroot -h192.168.63.128 -P9030
```

Hive：

```bash
beeline -u jdbc:hive2://192.168.63.128:10000
```

## 十、当前环境的关键约束

| 项目 | 说明 |
|---|---|
| 单节点部署 | 当前主要为单节点测试环境，不具备生产 HA |
| Kafka 单 Broker | 不适合高吞吐与副本验证 |
| Doris 单 BE | `replication_num` 必须为 `1` |
| ZooKeeper 单节点 | 不具备容灾能力 |
| Fluss 为预研形态 | 当前仅做轻量验证，不纳入一期主链路 |
