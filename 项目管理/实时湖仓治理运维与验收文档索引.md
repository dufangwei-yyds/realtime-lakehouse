# 实时湖仓治理运维与验收文档索引

> 文档定位：实时湖仓一期“数据治理、监控运维、项目验收”统一入口  
> 适用范围：一期实时湖仓 `ODS -> DIM -> DWD -> DWS -> ADS` 全链路治理、值守、发布、验收与交付归档

---

## 一、使用说明

本索引用于统一管理实时湖仓一期已经形成的治理运维与验收材料，便于：

1. 日常值守快速查阅
2. 发布与回滚时按文档执行
3. 项目结项与交付归档
4. 后续运维交接和持续维护

建议阅读顺序：

1. 先看主体方案
2. 再看规则与模板
3. 再看执行手册
4. 最后看验收材料
